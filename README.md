# Ürün İsmi
* Supplier Evaluation System

# Ürün Açıklaması
* Supplier Evaluation System, tedarikçi performansını ve risk seviyesini veri odaklı analiz eden yapay zekâ destekli bir karar destek platformudur. Sistem; finansal istikrar, teslimat performansı, kalite uyumluluğu, sürdürülebilirlik, geçmiş risk seviyesi ve ERP işlem geçmişi gibi kriterleri analiz ederek tedarikçilerin genel performansını değerlendirir. Kullanıcılar veri setlerini sisteme yükleyebilir, analiz sonuçlarını dashboard üzerinden görüntüleyebilir ve tedarikçiler arasında karşılaştırma yapabilir.

# Ürün Özellikleri
* Dashboard
* Veri yükleme (CSV / Excel)
* Veri önizleme
* Supplier listesi
* Supplier detay sayfası
* Risk kategorisi görüntüleme
* Genel performans skoru
* Hatalı dosya kontrolü
